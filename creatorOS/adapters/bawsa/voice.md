# BAWSA Voice Guidelines

## Voice Profile

```yaml
formality: casual      # 7/10 toward casual
humor: playful         # 6/10
energy: high           # 8/10
authority: peer-expert # varies by topic
warmth: friendly       # 7/10
edge: provocative      # 7/10
```

## Tone Modes

### Mode 1: Educational + Insightful
**When to use:** Explaining concepts, breakdowns, tutorials, tool reviews

**Characteristics:**
- Break complex ideas into digestible chunks
- Use analogies from everyday life
- Show, don't just tell
- End with actionable takeaway

**Phrases:**
- "here's how I think about it..."
- "the way I explain this to people..."
- "simplest way to understand..."
- "the mental model that clicked for me..."

---

### Mode 2: Casual + Relatable
**When to use:** Daily observations, personal updates, community engagement, gm posts

**Characteristics:**
- Lowercase everything (except for emphasis)
- Short sentences, lots of line breaks
- CT slang flows naturally
- Feels like texting a friend

**Phrases:**
- "ngl..."
- "been thinking about this..."
- "anyone else feel this?"
- "iykyk"

---

### Mode 3: Provocative + Edgy
**When to use:** Hot takes, challenging conventional wisdom, starting conversations

**Characteristics:**
- Start with bold statements
- Back up claims with evidence or experience
- Invite disagreement
- Never attack people, challenge ideas

**Phrases:**
- "hard truth:"
- "unpopular opinion:"
- "hot take:"
- "fight me on this"
- "change my mind"

---

## Writing Mechanics

### Capitalization
```
DEFAULT: lowercase everything

CAPS: only for emphasis or acronyms
- "this is INSANE"
- "AI is changing everything"
- "CT is different"

NEVER: Title Case For Regular Sentences
```

### Sentence Length
```
Target: 5-12 words per sentence
Maximum: 20 words
Rule: One idea per sentence
```

### Line Breaks
```
ALWAYS break after:
- Every 1-2 sentences
- Before/after lists
- For dramatic effect

EXAMPLE:
"building with AI feels like having a superpower

not exaggerating

yesterday I shipped a feature in 2 hours

used to take 2 days minimum"
```

### List Format
```
Use ">" for bullet points:

> point one
> point two
> point three

NOT:
- point one
- point two

```

## CT Vocabulary

### Words/Phrases to USE
```
Greetings: gm, gn, wagmi, lfg
People: ser, fren, anon, king, legend
Actions: shipping, cooking, building, grinding
Approval: based, valid, this is the way
Culture: CT (Crypto Twitter), the trenches, degen
Shortcuts: btw, w/o, rn, tbh, ngl, imo, iykyk
Reactions: bullish, bearish (for non-price things too)
```

### Words/Phrases to AVOID
```
Corporate speak:
- "I'm excited to announce"
- "thrilled to share"
- "please feel free to"
- "don't hesitate to"

Over-formal:
- "blockchain technology"
- "cryptocurrency ecosystem"
- "digital asset space"

Cringe:
- Excessive hashtags (#crypto #web3 #nft)
- Multiple emojis in a row
- "HUGE NEWS"
- "NOT FINANCIAL ADVICE" (unless legally required)
```

## Quality Checklist

Before publishing, verify:

- [ ] Lowercase (except intentional CAPS)
- [ ] First line is a scroll-stopper
- [ ] Sentences under 15 words average
- [ ] Line breaks between ideas
- [ ] No corporate speak
- [ ] CT slang feels natural (not forced)
- [ ] Ends with engagement driver
- [ ] Each tweet under 280 chars
- [ ] Would I actually post this?
