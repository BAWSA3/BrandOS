# BrandOS Scoring Framework — The Four Phases

```
 ┌─────────────────────────────────────────────────────────────────────────┐
 │                                                                         │
 │   BRAND SCORE = (DEFINE × 0.30) + (CHECK × 0.25) + (GENERATE × 0.25)  │
 │                 + (SCALE × 0.20) + Tier Bonus + Authority Bonus        │
 │                                                                         │
 │                                                          Max: 100       │
 └─────────────────────────────────────────────────────────────────────────┘
```


## The Pipeline

```
  ┌──────────┐      ┌──────────┐      ┌──────────┐      ┌──────────┐
  │          │      │          │      │          │      │          │
  │  DEFINE  │ ───► │  CHECK   │ ───► │ GENERATE │ ───► │  SCALE   │
  │          │      │          │      │          │      │          │
  │  30%     │      │  25%     │      │  25%     │      │  20%     │
  │          │      │          │      │          │      │          │
  └──────────┘      └──────────┘      └──────────┘      └──────────┘
   Who are you?     Are you           What are you      Is it
                    consistent?       putting out?      working?
```

## Phase 1: DEFINE — Brand Identity Clarity

```
  Weight: 30%
  ═══════════════════════════════════════════════════════════════

  QUESTION:  "Can someone understand what you're about
              from your content alone?"

  ┌─────────────────────────────────────────────────────────────┐
  │                                                             │
  │   WHAT IT MEASURES                                          │
  │   ─────────────────                                         │
  │   How clearly your brand identity comes through             │
  │   in your content, name, and reputation.                    │
  │                                                             │
  │   NOT measured: bio text (personal expression, not brand)   │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   HOW IT SCORES                                             │
  │   ─────────────                                             │
  │                                                             │
  │   Elite / Influential tier:                                 │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Does their reputation precede them?             │     │
  │   │ • Does the name alone carry brand weight?         │     │
  │   │ • Is content focused on clear themes or scattered?│     │
  │   │ • Do engagement patterns show a defined audience? │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   Emerging / Established tier:                              │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Does display name communicate the brand?        │     │
  │   │ • Is there a clear content focus or niche?        │     │
  │   │ • Do posting patterns suggest brand direction?    │     │
  │   │ • Is there a clear target audience from content?  │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   With tweets (enhanced):                                   │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Voice consistency across tweets                 │     │
  │   │ • Signature phrases or patterns                   │     │
  │   │ • Target audience clarity from content themes     │     │
  │   │ • Professional vs personal balance                │     │
  │   │ • Content focus and niche clarity                 │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   WHY IT MATTERS                                            │
  │   ──────────────                                            │
  │   A brand that can't be quickly understood can't grow.      │
  │   If people can't figure out what you're about from your    │
  │   content, they won't follow. Identity clarity is the       │
  │   foundation everything else builds on.                     │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   DATA INPUTS                                               │
  │   ───────────                                               │
  │   display name ─── username ─── follower count              │
  │   listed count ─── tweet volume ─── influence tier          │
  │   content themes (when tweets available)                    │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

## Phase 2: CHECK — Consistency & Credibility

```
  Weight: 25%
  ═══════════════════════════════════════════════════════════════

  QUESTION:  "Does everything about your brand feel
              like the same person?"

  ┌─────────────────────────────────────────────────────────────┐
  │                                                             │
  │   WHAT IT MEASURES                                          │
  │   ─────────────────                                         │
  │   Whether the brand feels cohesive and trustworthy          │
  │   across all touchpoints over time.                         │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   HOW IT SCORES                                             │
  │   ─────────────                                             │
  │                                                             │
  │   Profile signals:                                          │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Username ↔ Display name alignment               │     │
  │   │ • Tone consistency (does it feel cohesive?)       │     │
  │   │ • Verified badge (+10 credibility bonus)          │     │
  │   │ • Professional presentation quality               │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   Content signals (with tweets):                            │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Are they staying on-brand or drifting?          │     │
  │   │ • Topic focus consistency                         │     │
  │   │ • Any contradictory messaging?                    │     │
  │   │ • Voice consistency score (0-100)                 │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   Tier adjustments:                                         │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ Elite/Influential: consistency of reputation and  │     │
  │   │   content voice matters most                      │     │
  │   │ Emerging: everything working together as unified  │     │
  │   │   brand?                                          │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   WHY IT MATTERS                                            │
  │   ──────────────                                            │
  │   Inconsistency kills trust. If your name says one thing    │
  │   and your content says another, or your voice shifts       │
  │   wildly post to post, people can't form a mental model     │
  │   of who you are. Consistency compounds into reputation.    │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   DATA INPUTS                                               │
  │   ───────────                                               │
  │   username ─── display name ─── verified status             │
  │   voice spectrum analysis ─── content topic spread          │
  │   posting pattern regularity                                │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

## Phase 3: GENERATE — Content Quality & Output

```
  Weight: 25%
  ═══════════════════════════════════════════════════════════════

  QUESTION:  "Are you actually creating content that
              builds your brand?"

  ┌─────────────────────────────────────────────────────────────┐
  │                                                             │
  │   WHAT IT MEASURES                                          │
  │   ─────────────────                                         │
  │   The quality, volume, and effectiveness of content         │
  │   being produced.                                           │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   HOW IT SCORES                                             │
  │   ─────────────                                             │
  │                                                             │
  │   Profile-only mode:                                        │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Profile image present and professional?         │     │
  │   │ • Link present to drive traffic?                  │     │
  │   │ • Content output volume and consistency?          │     │
  │   │ • Using platform features (threads, media, polls)?│     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   Enhanced mode (with tweets):                              │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Engagement rate analysis (good for their size?) │     │
  │   │ • Content format effectiveness                    │     │
  │   │ • Hook quality (do tweets grab attention?)        │     │
  │   │ • CTA usage and effectiveness                     │     │
  │   │ • Posting frequency optimization                  │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   WHY IT MATTERS                                            │
  │   ──────────────                                            │
  │   Your brand is built through what you post. Volume         │
  │   creates surface area for discovery, quality determines    │
  │   whether people stay. A creator who doesn't generate       │
  │   content doesn't have a brand — they have a profile.       │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   DATA INPUTS                                               │
  │   ───────────                                               │
  │   profile image ─── website URL ─── tweet count             │
  │   posting frequency ─── engagement rates                    │
  │   content formats ─── media usage                           │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

## Phase 4: SCALE — Growth & Influence Signals

```
  Weight: 20%
  ═══════════════════════════════════════════════════════════════

  QUESTION:  "Is the market responding to the brand
              you're building?"

  ┌─────────────────────────────────────────────────────────────┐
  │                                                             │
  │   WHAT IT MEASURES                                          │
  │   ─────────────────                                         │
  │   Growth trajectory, audience quality, and influence        │
  │   reach — whether the market validates the brand.           │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   HOW IT SCORES                                             │
  │   ─────────────                                             │
  │                                                             │
  │   Follower/Following Ratio:                                 │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │  10:1+  ──── EXCELLENT (thought leader signal)    │     │
  │   │   2:1+  ──── Healthy                              │     │
  │   │   <2:1  ──── Room to grow                         │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   Listed Count (others curating you):                       │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │  10K+   ──── EXCEPTIONAL (high authority)         │     │
  │   │   1K+   ──── Strong curation signal               │     │
  │   │   <1K   ──── Building influence                   │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  │   Additional signals:                                       │
  │   ┌───────────────────────────────────────────────────┐     │
  │   │ • Growth trajectory from engagement patterns      │     │
  │   │ • Audience quality signals                        │     │
  │   │ • Viral potential (content type analysis)         │     │
  │   │ • Retweet patterns and reach                      │     │
  │   └───────────────────────────────────────────────────┘     │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   WHY IT MATTERS                                            │
  │   ──────────────                                            │
  │   Brand without reach is a diary. Scale signals show        │
  │   whether the market is responding to the brand being       │
  │   built. High listed count = others actively vouch for      │
  │   your value. Healthy ratio = people follow for content,    │
  │   not because you followed them first.                      │
  │                                                             │
  ├─────────────────────────────────────────────────────────────┤
  │                                                             │
  │   DATA INPUTS                                               │
  │   ───────────                                               │
  │   followers ─── following ─── ratio                         │
  │   listed count ─── tweet count                              │
  │   engagement rates ─── retweet patterns                     │
  │                                                             │
  └─────────────────────────────────────────────────────────────┘
```

## Influence Tier System

```
  Tier bonuses are added ON TOP of the base score.

  ┌──────────────┬───────────────┬────────────┬──────────────────────────┐
  │ TIER         │ FOLLOWERS     │ BONUS      │ EVALUATION LENS          │
  ├──────────────┼───────────────┼────────────┼──────────────────────────┤
  │ EMERGING     │ < 10K         │ +0         │ Optimization standards   │
  │ ESTABLISHED  │ 10K - 100K    │ +4         │ Growth positioning       │
  │ NOTABLE      │ 100K - 500K   │ +8         │ Next-level readiness     │
  │ INFLUENTIAL  │ 500K - 1M     │ +12        │ Brand effectiveness      │
  │ ELITE        │ 1M+           │ +15        │ Reputation & reach       │
  └──────────────┴───────────────┴────────────┴──────────────────────────┘

  Authority Bonus (stacks with tier):

  ┌────────────────────────────────────┬─────────────────────────┐
  │ SIGNAL                             │ BONUS                   │
  ├────────────────────────────────────┼─────────────────────────┤
  │ Follower/Following Ratio           │                         │
  │   100:1+                           │ +25                     │
  │   50:1+                            │ +20                     │
  │   10:1+                            │ +10                     │
  │   2:1+                             │ +5                      │
  ├────────────────────────────────────┼─────────────────────────┤
  │ Listed Count                       │                         │
  │   50K+                             │ +25                     │
  │   10K+                             │ +20                     │
  │   1K+                              │ +10                     │
  │   100+                             │ +5                      │
  ├────────────────────────────────────┼─────────────────────────┤
  │ Low tweet volume + high followers  │ +5                      │
  │ (selective, high-impact posting)   │                         │
  └────────────────────────────────────┴─────────────────────────┘
```

## Two Operating Modes

```
  ┌──────────────────────────────────┐     ┌──────────────────────────────────┐
  │     PROFILE-ONLY MODE            │     │     CONTENT-PRIMARY MODE         │
  │     (X API Free Tier)            │     │     (X API Basic Tier)           │
  ├──────────────────────────────────┤     ├──────────────────────────────────┤
  │                                  │     │                                  │
  │  Inputs:                         │     │  Inputs:                         │
  │  ├── Name, handle, PFP           │     │  ├── Everything in profile-only  │
  │  ├── Follower metrics            │     │  ├── 100 recent tweets           │
  │  ├── Listed count                │     │  ├── Per-tweet engagement stats  │
  │  ├── Verified status             │     │  ├── Posting cadence data        │
  │  ├── Tweet count (total only)    │     │  └── Content format breakdown    │
  │  └── Influence tier calc         │     │                                  │
  │                                  │     │  Extra analysis:                 │
  │  Limitations:                    │     │  ├── Voice consistency scoring   │
  │  ├── No content analysis         │     │  ├── Content theme detection     │
  │  ├── No engagement rates         │     │  ├── Signature phrase detection  │
  │  ├── No voice consistency        │     │  ├── Format performance data     │
  │  └── No topic detection          │     │  └── Hook quality analysis       │
  │                                  │     │                                  │
  │  Confidence: LOW                 │     │  Confidence: HIGH                │
  │                                  │     │                                  │
  └──────────────────────────────────┘     └──────────────────────────────────┘

  ┌──────────────────────────────────────────────────────────────────────────┐
  │                                                                          │
  │  IMPORTANT: Bio/description is NEVER a scoring factor in either mode.   │
  │  Bios are personal expression / taste — not brand signals.               │
  │  Brand is defined by CONTENT and how it correlates to audience growth.   │
  │                                                                          │
  └──────────────────────────────────────────────────────────────────────────┘
```

```
  ┌─────┬──────────────────────┬───────────────────────────────────────────┐
  │     │ ARCHETYPE            │ SIGNAL                                    │
  ├─────┼──────────────────────┼───────────────────────────────────────────┤
  │ 🎓  │ THE PROFESSOR        │ Deep knowledge, niche expert, educational │
  │ 🔌  │ THE PLUG             │ Super connector, networker, DMs open      │
  │ 🎪  │ CHIEF VIBES OFFICER  │ Entertainer, shitposter, meme energy      │
  │ 🔮  │ THE PROPHET          │ Visionary, strong opinions, called it     │
  │ 🚢  │ SHIP OR DIE          │ Builder, maker, product-focused           │
  │ 🐕  │ UNDERDOG ARC         │ Rising star, up-only trajectory           │
  │ 🎰  │ THE DEGEN            │ High-risk, crypto trader, ape mentality   │
  │ 👻  │ THE ANON             │ Pseudonymous legend, mysterious authority │
  └─────┴──────────────────────┴───────────────────────────────────────────┘
```

## Data Flow

```
                        ┌─────────────────┐
                        │   X Profile     │
                        │   API Fetch     │
                        └────────┬────────┘
                                 │
                    ┌────────────┴────────────┐
                    │                         │
                    ▼                         ▼
           ┌───────────────┐        ┌─────────────────┐
           │  Profile Data │        │  Tweet Data      │
           │  (always)     │        │  (if Basic tier) │
           └───────┬───────┘        └────────┬────────┘
                   │                         │
                   │    ┌────────────────────┤
                   │    │                    │
                   ▼    ▼                    ▼
           ┌──────────────────┐    ┌──────────────────┐
           │  Influence Tier  │    │  Tweet Voice      │
           │  Calculator      │    │  Analysis (Gemini)│
           │  (algorithmic)   │    │                   │
           └────────┬─────────┘    └────────┬─────────┘
                    │                        │
                    ▼                        ▼
           ┌─────────────────────────────────────────┐
           │                                         │
           │         Gemini LLM Scoring              │
           │                                         │
           │  Prompt includes:                       │
           │  ├── Profile metrics                    │
           │  ├── Influence tier + bonuses           │
           │  ├── Crypto/web3 signals (if detected)  │
           │  ├── Tweet analytics (if available)     │
           │  └── Sample tweets (if available)       │
           │                                         │
           │  Prompt EXCLUDES:                       │
           │  └── Bio/description text               │
           │                                         │
           └──────────────────┬──────────────────────┘
                              │
                              ▼
           ┌─────────────────────────────────────────┐
           │                                         │
           │         Scoring Output                  │
           │                                         │
           │  ├── DEFINE score (0-100)               │
           │  ├── CHECK score (0-100)                │
           │  ├── GENERATE score (0-100)             │
           │  ├── SCALE score (0-100)                │
           │  ├── Overall score (weighted + bonuses) │
           │  ├── Archetype assignment               │
           │  ├── Top strengths (3)                  │
           │  ├── Top improvements (3)               │
           │  └── Summary assessment                 │
           │                                         │
           └─────────────────────────────────────────┘
```
