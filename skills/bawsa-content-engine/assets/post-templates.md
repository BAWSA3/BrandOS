# Post Templates

## Thread Template

```
1/ [bold claim or surprising stat]

[1-2 lines expanding on why this matters]

[thread indicator: "here's what I found:" or "a thread:"]

2/ [first point — the most interesting one]
[2-3 lines of proof, example, or explanation]

3/ [second point]
[2-3 lines]

4/ [third point]
[2-3 lines]

5/ [fourth point — the most actionable one]
[2-3 lines]

6/ [summary or key takeaway]
[tie it back to the hook]

7/ [CTA — always RT tweet 1 + follow]
RT tweet 1 if this was useful.
follow @BawsaXBT for more on [topic].
```

## Framework Post Template

```
[framework name] that [result/outcome]:

[item 1] ([brief description])
[item 2] ([brief description])
[item 3] ([brief description])

[frequency or application note]

[CTA: save this if you [problem it solves]]
```

## Hot Take Template

```
[unpopular opinion or bold claim]

[2-3 lines of reasoning — why you believe this]

[optional: what most people do wrong instead]

[CTA: agree or disagree?]
```

## Build Log Template

```
[time stamp]. [what you're doing].

[what you just shipped/built/discovered]

[1-2 lines on why it matters or what you learned]

[CTA: reply with what you're building / reply with yours]
```

## Data Post Template

```
[analyzed/tracked/measured] [what]:

- [stat 1]
- [stat 2]
- [stat 3]

[insight — what the data means]

[CTA: tag someone / save this]
```

## Community Post Template

```
[day] [vibe check/builder check-in/question]:

[the question or prompt — specific, not generic]

[optional: why you're asking or what prompted it]

[CTA: drop your handle / drop it below / reply with yours]
```

## Quote RT Bait Template

```
[your take or experience on a topic]

[1-2 lines expanding]

[CTA: quote this with your [version/experience/take]]
```

## Value Offer Template (1x per week)

```
[what you're offering — be specific]

reply "[keyword]" and I'll [specific deliverable] for [number] people.

[credibility line — why you're qualified]

[CTA: the reply keyword]
```

## Week Recap Template (Sunday Post 2)

```
week [X] of [system/journey]:

- [win 1]
- [win 2]
- [win 3]
- [learning or insight]

[1-line summary of what you learned]

[CTA: follow if you want to see [what's next]]
```
